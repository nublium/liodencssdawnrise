## Files

| File | Purpose |
|---|---|
| `dawnrise-den.css` | The complete stylesheet — page chrome, chat, tables, the territory layout. |
| `dawnrise-den.html` | The territory-description snippet, with copy-paste-ready placeholders and inline comments explaining what each block does. |
| `dawnrise-preview.html` | Standalone preview that simulates the Lioden page around the territory. Open it locally to see exactly how it will render. |
| `placeholder-dawnrise-avatar.svg` | Placeholder avatar — round sun-lion crest. Replace with your own art. |
| `placeholder-cave.svg` / `placeholder-mound.svg` | Thumbnails that swap in for Lioden's default cave/mound tile art. |
| `placeholder-unsorted.svg` / `placeholder-nesting.svg` / `placeholder-pride.svg` | Themed banners for the three organisation tiles at the top of the cave list. |

## How to use it

1. Open your Lioden territory description editor.
2. Copy the **entire** contents of `dawnrise-den.html` (including the `<link>` line).
3. Paste it into the description box and save.
4. Edit only the text inside the blocks marked `[YOUR TEXT HERE]` — the CSS handles all the visual styling.

The CSS is served from this repo via GitHub Pages, so updates to the look
ship automatically.
